
h1. e9s

Enrichments (e9s) for Ruby Internationalization (i18n)

h2. Introduction

E9s was created due to the need of simply implementing i18n within a Rails application. It enriches I18n, Formtastic, the String and Symbol classes to make life easier for the Rails developer. Want to see what Enrichments (e9s) really has to offer within a couple of seconds? Please visit "http://github.com/archan937/e9s-demo":http://github.com/archan937/e9s-demo for a complete demo application.

h2. Installation

h3. Using e9s as gem

Add gemcutter as a gem source, if you haven't already done it:

<pre>
  sudo gem sources -a http://gemcutter.org/
</pre>

Install the e9s gem:

<pre>
  sudo gem install e9s
</pre>

Add e9s in environment.rb as a gem dependency:

<pre>
  config.gem "e9s"
</pre>

h3. Using e9s as plugin

Install the e9s plugin:

<pre>
  ./script/plugin install git://github.com/archan937/e9s.git 
</pre>

Optionally, you can also install the Formtastic plugin:

<pre>
  ./script/plugin install git://github.com/justinfrench/formtastic.git
</pre>
